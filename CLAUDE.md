## Project Overview

This is a Python project using uv for package management.

## Development Commands

```bash
# Run the main script
uv run python main.py

# Add a dependency
uv add <package-name>

# Sync dependencies
uv sync
```

## Requirements

- Python 3.12+
- uv package manager
